# 🤖 AI 롱폼 스토리 생성기

AI 기반 유튜브 롱폼 콘텐츠 스토리 자동 생성 도구

## 🎯 **NEW! 이중 모드 시스템**

### 🚀 간단 모드 (Simple Mode)
- ⚡ **원클릭 생성**: AI가 모든 요소를 랜덤 조합
- 🎲 **25,200가지 조합**: 10 장르 × 15 배경 × 14 직업 × 12 이슈
- 🕐 **2-3분 완성**: 빠른 아이디어 도출
- 👤 **초보자 친화적**: 복잡한 설정 없이 바로 시작

### 🎛️ 고급 모드 (Advanced Mode)  
- 🏗️ **복잡한 시간 구조**: 순차적/회상/평행/순환/퍼즐 구조
- 🎨 **다양한 서술 방식**: 3인칭/1인칭/다중시점/다큐멘터리
- 🎭 **핵심 테마 설정**: 10가지 깊이 있는 주제
- 🔧 **전문가급 제어**: 모든 요소 세밀 조정

## 🚀 주요 기능

### 🎮 이중 모드 시스템
**간단 모드**: 25,200가지 자동 조합으로 초고속 생성  
**고급 모드**: 무한 변주 가능한 정교한 커스터마이징

### 📖 혁신적 스토리 구조
- **순차적 구조**: 자연스러운 시간 순서 전개
- **회상 구조**: 현재→과거→현재 미스터리 강화
- **평행 구조**: 여러 시점 동시 진행으로 입체감
- **순환 구조**: 끝이 시작으로 연결되는 완결성  
- **퍼즐 구조**: 조각조각 맞춰가는 추리적 재미

### 🎭 다중 서술 방식
- **3인칭 관찰자**: 영화적 시각 서술
- **1인칭 주인공**: 생생한 내적 독백  
- **다중 시점**: 여러 인물 관점 교차
- **다큐멘터리**: 객관적 현실감

### 🔄 무한 변주 시스템
- **간단 모드**: 10 장르 × 15 배경 × 14 직업 × 12 이슈 = **25,200가지**
- **고급 모드**: 5 시간구조 × 4 서술방식 × 10 테마 × 무한 커스텀 = **무한대**

### API 최적화
- 단계별 분할 처리로 효율적인 대본 생성
- Claude Sonnet 4 API 연동으로 고품질 스토리텔링
- 모드별 차별화된 프롬프트 시스템

### 롱폼 특화  
- 30분~2시간 분량 자유 조절
- 3-10개 반전 포인트 자동 배치
- 시청자 이탈 방지 구조 설계

## 🚀 설치 및 실행

### 1. 저장소 클론
```bash
git clone https://github.com/yourusername/ai-longform-story-generator.git
cd ai-longform-story-generator
```

### 2. 간단 실행 (권장)
```bash
# Windows
start.bat

# Mac/Linux  
chmod +x start.sh
./start.sh
```

### 3. 수동 실행
```bash
npm install
npm run dev
# http://localhost:3000 또는 자동 할당 포트로 접속
```

## 📝 사용법

### 🎮 모드 선택
1. **간단 모드**: 설정 최소화, 랜덤 조합으로 빠른 생성
2. **고급 모드**: 모든 요소 커스터마이징, 전문가급 제어

### 📋 단계별 과정
1. **모드 선택**: 간단 vs 고급 모드
2. **기본 설정**: 분량(30-120분), 캐릭터 수(3-6명), 반전 포인트(3-10개)
3. **세부 설정** (고급 모드만):
   - 장르, 배경, 주인공 직업, 사회적 이슈
   - 시간 구조, 서술 방식, 핵심 테마
4. **AI 생성**: 컨셉 → 플롯 → 첫 챕터
5. **다운로드**: 완성된 대본 + 추가 생성 프롬프트

## 🎮 사용 예시

### 간단 모드 예시
```
설정: 60분, 4캐릭터, 5반전
결과: 전직 해커 + 요양병원 + 노인학대 → 12챕터 스릴러
시간: 2-3분 완성
```

### 고급 모드 예시  
```
장르: 미스터리 | 배경: 방송국 | 직업: 프로파일러
시간구조: 회상 | 서술: 다중시점 | 테마: 진실과 거짓
결과: 복합적 시간축의 정교한 심리 스릴러
시간: 3-4분 완성
```

## 🔧 고급 설정 옵션

### 시간 구조
- **순차적**: 일반적인 시간 순서 (A→B→C)
- **회상**: 현재→과거→현재 (C→A→B→C)  
- **평행**: 여러 시점 동시 (A1→B1→A2→B2)
- **순환**: 끝=시작 (A→B→C→A')
- **퍼즐**: 무작위 조각 (B→A→D→C)

### 서술 방식
- **3인칭 관찰자**: "그는 문을 열었다"
- **1인칭 주인공**: "나는 문을 열었다"  
- **다중 시점**: 여러 인물의 1인칭 교차
- **다큐멘터리**: 객관적 기록 방식
