# ⚡ Vercel 원클릭 배포 가이드

## 🚀 3분만에 온라인 링크!

### 방법 1: 드래그 앤 드롭
1. **프로젝트 빌드**:
   ```cmd
   cd C:\Users\user\Desktop\longform-story-generator
   npm run build
   ```

2. **https://vercel.com** 접속
3. **"Deploy"** 버튼 클릭  
4. **dist 폴더** 드래그 앤 드롭
5. **즉시 링크 생성!**

### 완성 링크 예시:
```
https://ai-longform-story-generator.vercel.app/
```

### 방법 2: GitHub 연동 (자동 업데이트)
1. GitHub에 업로드 (이전 가이드 참조)
2. Vercel에서 GitHub 저장소 연결
3. 자동 배포 및 업데이트
